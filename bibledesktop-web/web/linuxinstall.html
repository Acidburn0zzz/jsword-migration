Getting Bible Desktop to work on Fedora Core 3.

I) The first thing is to get a version of Java that will run Bible Desktop.

If you have not done so, follow these instructions:

  • Obtain and install Java

II) At this point you have can get Bible Desktop and run it.

At this point and time Java's WebStart does not work under Fedora Core 3. This is a reported bug in Sun's Java (see Sun's BugParade ID# 6188963) and will be fixed in some point release of Java 1.5. For this reason, Bible Desktop will need to be manually installed and run.

  1. Download JSword from the JSword download page. This contains both Bible Desktop and the JSword librarys that it is built on top of. Pick any of the *.tar.gz binary packages.
  2. Unpack the package into a directory of your choice. The jsword directory will be created for you.
  3. cd into that directory and execute jsword.sh. This is the Bible Desktop program.
