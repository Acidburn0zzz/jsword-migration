CVS Access

CVS is hosted at crosswire. The CVS root is:
:pserver:anonymous@cvs.crosswire.org:/cvs/jsword
And the password is "anonymous". If you need to get at the code but are not sure about CVS then join the mailing list and we'll sort something out. The module to checkout is 'jsword', so if you are using the command line you would do something like this (having first set your CVSROOT:
cvs checkout jsword.
There is also a support module containing 3rd-party source jars, tools to build the website, UML models and a default KJV module. To get access to this you will need to check-out jsword-support.

For detailed instruction on how to use Eclipse as a CVS client see: Using Eclipse with JSword
