Source Code Access

This is currently under construction. Some info is inaccurate.

Our source code is managed with Subversion, aka SVN. SVN is hosted at crosswire. The SVN root is:
https://crosswire.org/svn/jsword
Anyone can access the code without a password and can be browsed at this url.
The project to checkout is jsword, so if you are using the command line you would do something like this
svn checkout https://crosswire.org/svn/jsword/trunk/jsword jsword.
You will need to repeat this for each of bibledesktop, common and common-swing. There is also jsword-support, containing 3rd-party QA tools
jsword-web contains the code for the JSword website and bibledesktop-web houses the website for BibleDesktop.

For detailed instruction on how to use Eclipse as a SVN client see: Using Eclipse with JSword
