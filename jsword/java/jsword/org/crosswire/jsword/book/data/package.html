The data package manages the book data returned by the various books.

The core interfaces are BookData which is returned by many of the methods on Book (and it's subclasses). BookData works with nested SectionData and VerseData objects to give a very high level overview of the text within an instance of BookData.

Construction of a BookData is does via an implementation of BookDataListener. (These implementations are created using DataFactory)
