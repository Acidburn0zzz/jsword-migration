Eireneh.com - JSword

About

JSword is a platform for various Bible programs using Java. It will support desktop applications like the On-Line Bible along with other interfaces like servlet, applet, etc.

JSword is split into several sections, a package for managing verses and lists of verses, a package to allow access to Bible text, packages to allow searching and several GUI sections.
